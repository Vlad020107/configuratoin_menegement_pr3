Язык CSV-скриптов
Данный репозиторий представляет собой спецификацию простого скриптового языка, основанного на синтаксисе CSV (Comma-Separated Values). Основная цель языка — выполнение простых операций над числами и строками с использованием встроенных функций, выполняемых последовательно по строкам файла.

Формат описания
Формально каждая команда представлена в виде строки CSV-файла. Первая строка является заголовком и определяет типы команд, используемых в последующих строках. Каждая последующая строка соответствует одной команде и её аргументам.

Семантика языка
Шапка:
*const — создание постоянной переменной (константы).
*reading — чтение данных из внешнего хранилища (например, файлового ресурса).
*writing — запись данных во внешний ресурс.
*sgn — вычисление знака числа (унарная операция).
Столбцы значений:
Каждая строка описывает либо операцию, либо непосредственно обрабатываемое значение. Значения интерпретируются согласно заданному типу операции в первой строке («шапке»).

Прагматика языка
*Константа (const): позволяет задать фиксированное значение для дальнейшего использования.
*Операция чтения (reading): извлекает значение из указанного места хранения (например, файла).
*Запись (writing): сохраняет указанное значение в файле или другом ресурсе.
*Унарная операция (sgn): возвращает знак числа (+1, -1 или 0 соответственно).
